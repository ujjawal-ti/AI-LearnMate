Test Page for Tool Assistant Extension

About This Page

This is a test page to verify that the Tool Assistant Chrome extension is working properly. The extension should inject a floating chat button in the bottom-right corner of this page.

When you click the chat button, it should open an inline chat widget that allows you to ask questions about the content of this page.

Features to Test:

  • Floating chat button visibility
  • Chat widget opening and closing
  • Page content extraction
  • AI assistant responses

Sample Content

This page contains information about web development, Chrome extensions, and AI assistants. You can ask the assistant questions like:

  • "What is this page about?"
  • "What features are being tested?"
  • "Tell me about Chrome extensions"

The Tool Assistant uses advanced AI to understand the content of web pages and provide helpful responses based on that content.

Technical Details

The Tool Assistant extension consists of:

  • Content Script: Injects the chat interface into web pages
  • Background Script: Handles extension lifecycle events
  • Backend API: FastAPI server with AI capabilities
  • Vector Search: FAISS-based document retrieval

The system can extract and analyze webpage content, then provide contextual responses using a combination of the current page content and a knowledge base.
